What the MVP is

A human-supervised Rails web application that converts batches of global invoices into normalized, validated, reviewable, export-ready accounting data.

  1. Upload PDF/image/structured invoice files.
  2. Inspect, normalize, and route safely.
  3. During M2.5, add Docling/native extraction, PaddleOCR-VL, and Qwen3-VL behind the completed M2 provider contract.
  4. Validate with deterministic Ruby services.
  5. Review flagged fields against source evidence.
  6. Approve an immutable revision.
  7. Export approved JSON, CSV, or XLSX.

Open-source model stack

Qwen3-VL-4B-Instruct

Canonical semantic extraction. Exact model revision and quantization are recorded.

PaddleOCR-VL-1.6

Full layout/OCR pipeline for scans, tables, reading order, and evidence boxes.

Docling

Native digital PDF text, page structure, and table parsing.

Ruby validators

Final authority for money, totals, duplicates, blocking findings, and acceptance.
