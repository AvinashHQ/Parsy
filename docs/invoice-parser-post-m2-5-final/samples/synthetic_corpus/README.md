# Synthetic Invoice Corpus

This folder contains 29 generated functional and negative fixtures for the post-M2 invoice parser.

Start with `manifest.csv`. Each row identifies the source file, expected route/status, Canonical Invoice v2 ground truth where applicable, deterministic expected findings, and SHA-256.

All names, identifiers, addresses, and transactions are fictitious. The corpus may be used and modified for this project's testing. It must not be represented as real customer evidence or as a statistically valid model benchmark.

See `../../docs/20_SYNTHETIC_CORPUS_GUIDE.md` for test suites and fixture descriptions.
