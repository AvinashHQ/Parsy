# First-Live Format Support Matrix — M0 Frozen

| Capability | Status at M1 start | First-live claim |
|---|---|---|
| Visual PDF | Required | Benchmarked after M2 |
| JPEG/PNG/TIFF | Required | Benchmarked after M2 |
| English/Latin script | Required | Named pilot profile only |
| Multiple countries | Represented | No universal compliance claim |
| Multiple ISO currencies | Required | Precision validated in M1 |
| Invoice | Primary | Yes after benchmark |
| Credit note | Primary | Yes after benchmark |
| Receipt/debit note | Review-required | Limited pilot support |
| Generic tax breakdown | Required | Source representation and arithmetic only |
| Line items | Conditional | Pilot-specific, human-reviewed |
| Factur-X/ZUGFeRD/UBL/CII | Detect/quarantine | No extraction claim in first live MVP |
| Regional tax validation | Disabled | No |
| Direct ERP posting | Out of scope | No |

## Unsupported first-live cases

Non-English primary text, non-Latin scripts, handwriting-dominant documents, statements, purchase orders, delivery notes, pro-forma invoices, government submission and any unnamed structured profile.
